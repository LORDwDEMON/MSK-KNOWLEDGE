# Incident # (Инцидент безопасности)

## 1. Назначение сущности

Сущность Incident предназначена для регистрации, учета, анализа и управления событиями, связанными с нарушением требований безопасности, возникновением опасной ситуации, причинением вреда здоровью работников либо потенциальной возможностью возникновения негативных последствий.

Сущность является центральным объектом домена DOM-SAFETY для управления:

- происшествиями;
- опасными событиями;
- нарушениями требований охраны труда;
- случаями воздействия опасных факторов;
- событиями, требующими расследования и корректирующих мероприятий.
